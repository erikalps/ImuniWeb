# 💉 Sistema de Consulta de Vacinas por Faixa Etária

## 📌 Sobre o projeto
Este projeto é uma **interface digital interativa** que permite aos usuários consultarem as vacinas obrigatórias e recomendadas com base em sua faixa etária.

A aplicação foi desenvolvida com foco em facilitar o acesso à informação sobre vacinação, proporcionando uma experiência simples, clara e intuitiva para o usuário.

---

## 🚀 Funcionalidades

- Inserção da idade do usuário  
- Exibição das vacinas recomendadas conforme a faixa etária  
- Informações detalhadas sobre cada vacina:
  - Descrição  
  - Idade recomendada  
  - Prazo para aplicação  
  - Local de aplicação  
- Interface interativa e de fácil uso  
- Feedback visual para o usuário  

---

## 🛠️ Tecnologias utilizadas

- **HTML5** – estrutura da aplicação  
- **CSS3** – estilização e layout  
- **JavaScript** – lógica da aplicação e interatividade
